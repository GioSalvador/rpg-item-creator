**RPG Item Creator** is a front-end application built with Next.js and Tailwind, intended to allow users to design and generate RPG items via a user-friendly interface.

## Features

- Item creation UI: define name, image, stats, value, etc.  
- Instant preview and editing capabilities.  
- Responsive front-end design (via Tailwind CSS)  
- Strong TypeScript support.

## Getting Started (Local Setup)

### Prerequisites

- Node.js (v14+ recommended)  
- npm / yarn / pnpm (choose your preferred package manager)  
- Git (for cloning the repository)

1. Clone the repository:  
   ```bash
   git clone https://github.com/GioSalvador/rpg-item-creator.git
   cd rpg-item-creator
   
2. Install dependencies: 
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   
3. Start the development server: 
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   
4. Open http://localhost:3000 in your browser to view the app.

## Usage

Once the server is running, use the UI to:
- Fill in the item fields: name, type, rarity, stats, image, etc.
- Preview the item display immediately.
